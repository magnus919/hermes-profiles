# Frontend Engineer — Hermes Profile

Frontend engineer — implements UI components, state management, API integration, responsive layouts, and client-side performance.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/frontend-engineer ~/.hermes/profiles/
hermes --profile frontend-engineer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| `artifact-pyramids` | Progressive disclosure output format |
| `frontend-engineering` | Component architecture, state management, API integration, performance methodology |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
