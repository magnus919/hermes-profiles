# Brand Designer — Hermes Profile

Brand identity systems — strategy, visual identity (logo, color, typography, imagery), voice/tone, and usage guidelines.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/brand-designer ~/.hermes/profiles/
hermes --profile brand-designer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| artifact-pyramids | Output format |
| brand-designer | Brand strategy, visual identity, color systems, typography, imagery, templates |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
