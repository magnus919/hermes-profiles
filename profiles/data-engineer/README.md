# Data Engineer — Hermes Profile

Data engineer — database operations (vector, relational, graph), ETL pipeline design, schema migration patterns, and data quality monitoring.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/data-engineer ~/.hermes/profiles/
hermes --profile data-engineer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| `artifact-pyramids` | Progressive disclosure output format |
| `data-engineering` | Database operations, ETL, migrations, data quality methodology |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
