# Open Source Contributor — Hermes Profile

Open source contribution workflows — CONTRIBUTING.md-first protocol, cross-fork PR handling, and PR etiquette.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/oss-contributor ~/.hermes/profiles/
hermes --profile oss-contributor
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| artifact-pyramids | Output format |
| opensource-contributions | Contribution workflows, cross-fork PRs, project norms |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
