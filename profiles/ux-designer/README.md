# Ux-Designer — Hermes Profile

UX designer — maps user journeys, designs interaction flows, applies accessibility standards, and ensures product decisions account for human experience.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/ux-designer ~/.hermes/profiles/
hermes --profile ux-designer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| `artifact-pyramids` | Progressive disclosure output format |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
