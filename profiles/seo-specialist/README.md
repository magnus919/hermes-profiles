# SEO Specialist — Hermes Profile

Search engine optimization — technical SEO, content optimization, metadata management, and search performance analysis.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/seo-specialist ~/.hermes/profiles/
hermes --profile seo-specialist
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| artifact-pyramids | Output format |
| seo-audit | Technical SEO, on-page SEO, schema markup, content strategy |
| seo-content-optimization | Keyword research, content gaps, search intent alignment |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
