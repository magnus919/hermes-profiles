# Security Engineer - Hermes Profile

Security engineer - threat modeling, vulnerability assessment, security architecture review, and penetration testing methodology.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/security-engineer ~/.hermes/profiles/
hermes --profile security-engineer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| `artifact-pyramids` | Progressive disclosure output format |
| `security-audit-methodology` | Threat modeling, vulnerability assessment, security architecture review |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
