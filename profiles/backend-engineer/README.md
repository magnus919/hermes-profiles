# Backend Engineer — Hermes Profile

Backend engineer — implements API endpoints (REST/gRPC/GraphQL), service logic, database access patterns, integration code, and service-level observability.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/backend-engineer ~/.hermes/profiles/
hermes --profile backend-engineer
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| `artifact-pyramids` | Progressive disclosure output format |
| `backend-engineering` | API patterns, service architecture, database access, integration, error handling methodology |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
