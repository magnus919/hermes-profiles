# Editor — Hermes Profile

Structural editing, argument coherence, engagement quality, and voice consistency for published content.

## Installation

```bash
git clone https://github.com/magnus919/hermes-profiles.git ~/hermes-profiles
ln -s ~/hermes-profiles/profiles/editor ~/.hermes/profiles/
hermes --profile editor
```

## Skill Dependencies

| Skill | Provides |
|---|---|
| artifact-pyramids | Output format |
| editor-methodology | Editorial process, revision protocol |
| editor-review-methodology | Fact-checking, voice audit, engagement review, source integrity |

## Output Format

Artifact pyramid. Response is the absolute path to `00-index.md`.
