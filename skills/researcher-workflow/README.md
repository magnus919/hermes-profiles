# researcher-workflow Bundle

Generated by workflow-architect on 2026-05-31.

This bundle encodes the researcher subagent's workflow: a non-interactive deep research pipeline that takes an orchestrator's mission brief and produces a progressive disclosure artifact pyramid.

## Skills

| Skill | Trigger | When it fires |
|-------|---------|---------------|
| receive-mission | Orchestrator assigns a research brief | Entry point — reformulates brief into structured scope |
| research-gather | After scope is defined | Executes multi-pass research using groktocrawl exclusively |
| evaluate-gaps | After initial gather | Applies three-question model, decides recursion |
| build-pyramid | After saturation reached | Assembles layered artifact files in /tmp/ |
| deliver-findings | After pyramid built | Reports paths back to orchestrator with layer descriptions |

## Loading

The bundle is loaded automatically when the researcher profile is dispatched. To load a specific sub-skill: `skill_view(name='researcher-workflow/<skill-name>')`

## Key Principles

- **No web_search or web_extract.** ALL web research uses groktocrawl.
- **Progressive disclosure.** Each layer of the pyramid links downward with descriptions.
- **Three-question gap model.** In-scope? Changes conclusions? Depth or bulk?
- **OS-level cleanup.** Artifacts in /tmp/ expire naturally.
