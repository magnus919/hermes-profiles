# Accessibility Checklist

## WCAG 2.1 Level AA Minimum

### Perceivable
- [ ] All non-text content has text alternatives
- [ ] Captions provided for all audio content
- [ ] Content can be presented without loss of meaning when CSS is disabled
- [ ] Color is not the only means of conveying information
- [ ] Text contrast ratio meets 4.5:1 minimum
- [ ] Non-text contrast meets 3:1 minimum
- [ ] Text can be resized up to 200% without loss of functionality

### Operable
- [ ] All functionality is available via keyboard
- [ ] No keyboard traps
- [ ] Focus order is logical and meaningful
- [ ] Focus indicators are visible
- [ ] Timed responses can be extended or disabled
- [ ] No content flashes more than 3 times per second
- [ ] Skip navigation links are provided

### Understandable
- [ ] Language of the page is specified
- [ ] UI components have consistent labels and instructions
- [ ] Error messages are descriptive and suggest corrections
- [ ] Input validation provides clear error identification

### Robust
- [ ] HTML elements have complete start/end tags
- [ ] ARIA attributes are used correctly
- [ ] Custom components have appropriate roles and states
