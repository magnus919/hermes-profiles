# Editorial Review Methodology

Review dimensions for assessing whether a draft is ready for publication.

## Review Dimensions

| Dimension | Question | Reference |
|-----------|----------|-----------|
| **Fact-check** | Are all factual claims verifiable? | `references/fact-checking.md` |
| **Voice audit** | Is the tone consistent throughout? | `references/voice-audit.md` |
| **Engagement** | Does the piece earn the reader's attention? | `references/engagement-review.md` |
| **Source integrity** | Are sources properly attributed and verified? | `references/source-integrity.md` |

## Review Protocol

1. **Structural first, line-by-line last.** Don't proofread until the structure is solid.
2. **Flag and suggest — don't rewrite.** The editor identifies issues; the writer decides how to fix them.
3. **One dimension per pass.** Don't mix fact-checking with voice audit in the same reading.
