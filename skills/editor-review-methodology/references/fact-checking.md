# Fact-Checking Protocol

## Claim Types and Verification

| Claim Type | Verification Standard | Example |
|-----------|----------------------|---------|
| **Statistical** | Must cite primary source. Check sample size, methodology, date. | "42% of users prefer..." |
| **Historical** | Must cite date and source. Cross-check with second source. | "The company was founded in 2019" |
| **Technical** | Must be reproducible or cite authoritative reference. | "The API returns 200 on success" |
| **Attributed quote** | Must match original source verbatim. | "As Jane Doe said..." |

## Standard Questions for Every Claim

1. Where does this information come from?
2. Is the source authoritative on this topic?
3. Is the source up to date?
4. Is the claim accurately represented from the source?
5. Is the claim in scope for what the source actually says?

## Unverifiable Claims

If a claim cannot be verified against available sources:
- Flag it explicitly
- Suggest removing it or qualifying it
- Never approve a draft with unverifiable factual claims
