# Voice Audit

## Purpose

Assess whether the draft maintains a consistent voice register throughout. Flag tone shifts, jargon creep, and register breaks.

## Common Voice Issues

| Issue | Example | Fix |
|-------|---------|-----|
| **Register shift** | Starting conversational, switching to academic mid-piece | Pick one register and apply throughout |
| **Jargon creep** | Technical terms appearing without definition | Define on first use or remove if unnecessary |
| **Tone inconsistency** | Humorous passage in an otherwise serious piece | Match tone to the piece's purpose and audience |
| **Person shift** | Switching between first/second/third person | Pick one and be consistent |

## Audit Protocol

1. Read the entire draft once without taking notes — feel where the voice breaks
2. Second pass: flag specific sections with register shifts
3. For each flag, state: what register the section is in vs. what register it should be in
4. Suggest specific rewrites, not general feedback
