# Source Integrity

## Purpose

Verify that the draft properly attributes its sources and treats them honestly.

## Attribution Standards

| Source Type | Required Attribution | 
|-------------|---------------------|
| Directly quoted | Name of speaker, source of quote |
| Paraphrased from specific source | Named source, linked if available |
| Common knowledge | No attribution required |
| Statistical claim | Named source, link to primary data |
| Derived analysis | Clarify that analysis is the author's, based on cited data |

## Integrity Violations

| Violation | Description |
|-----------|-------------|
| **Over-attribution** | Citing a source that doesn't actually support the claim |
| **Cherry-picking** | Quoting only the part that supports the argument while ignoring contradicting context |
| **False equivalence** | Presenting two sources as equally authoritative when one is clearly more credible |
| **Anonymous laundering** | Using anonymous sources to make claims that should be attributed |

## Checklist

- [ ] Every factual claim has a source
- [ ] Sources are primary where possible
- [ ] Quotes are accurate and in context
- [ ] Opposing views are fairly represented
- [ ] The author's own analysis is clearly distinguished from sourced claims
