# Copy-Editing Methodology

Line-level editing and proofreading. Does not restructure or rewrite — the editor handles argument and voice. The copy-editor handles the surface.

## The Copy-Editing Protocol

```
STYLE → GRAMMAR → CONSISTENCY → PROOFREAD
```
