# Copy-Editing Checklist

## Pass 1: Grammar and Punctuation

- [ ] Subject-verb agreement
- [ ] Pronoun-antecedent agreement
- [ ] Comma usage (serial commas, restrictive vs non-restrictive clauses)
- [ ] Apostrophe usage (possessives vs plurals)
- [ ] Consistent quotation mark style
- [ ] Hyphen usage in compound modifiers

## Pass 2: Consistency

- [ ] Consistent spelling (US vs UK English)
- [ ] Consistent capitalization of proper nouns and terms
- [ ] Consistent formatting (date formats, number formats)
- [ ] Consistent terminology (same term used the same way throughout)
- [ ] Consistent heading hierarchy (no skipped levels)

## Pass 3: Clarity

- [ ] Run-on sentences flagged
- [ ] Ambiguous pronoun references flagged
- [ ] Jargon defined on first use
- [ ] Abbreviations expanded on first use
- [ ] Parallel structure in lists and comparisons

## Pass 4: Mechanics

- [ ] All links resolve and are formatted correctly
- [ ] Code blocks are syntactically valid
- [ ] Images have alt text
- [ ] Footnotes/endnotes are sequential
- [ ] Citations match the reference list
