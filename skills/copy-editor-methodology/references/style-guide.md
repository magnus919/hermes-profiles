# Style Guide Reference

## Common Style Decisions

| Decision | Option A | Option B | Default |
|----------|----------|----------|---------|
| Serial comma | Use it (Oxford) | Skip it | Use it |
| Em dashes | Allow with spaces | Allow without spaces | Replace with semicolons/commas |
| Numbers | Spell out under 10 | Use digits for all | Spell out under 10 |
| Quotation marks | Logical punctuation | Typesetter's punctuation | Logical (periods outside quotes unless part of quoted material) |
| Headings | Title case | Sentence case | Sentence case |
| Hyphenation | Close compounds (email) | Open compounds (e mail) | Follow dictionary |
| Contractions | Allow in informal | Forbid in all | Allow in conversational register |

## Style Guide Enforcement

When enforcing a style guide:
1. Identify which style guide applies (house style, Chicago, AP, etc.)
2. Flag every deviation — don't decide which ones "matter"
3. Be consistent within the decision. If serial commas are used, use them everywhere.
4. Document the convention at the top of the findings report
