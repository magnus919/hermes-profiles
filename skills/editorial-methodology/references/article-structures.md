# Article Structures

## Common Patterns

| Structure | Best For | Flow |
|-----------|----------|------|
| **Inverted pyramid** | News, announcements | Most important info first → supporting details → background |
| **Problem-solution** | Technical tutorials, case studies | Describe the problem → present the solution → show the results |
| **Listicle** | Reference material, comparisons | Each item is self-contained, ordered by importance |
| **Narrative arc** | Story-driven pieces, personal essays | Setup → conflict → resolution → reflection |
| **Argumentative** | Opinion pieces, analysis | Thesis → evidence → counter-argument → conclusion |

## Structural Discipline

1. **Outline before writing.** Every piece has a skeleton before it has flesh.
2. **One idea per paragraph.** If a paragraph makes two points, split it.
3. **Section headings tell the story.** A reader who skims the headings should understand the argument.
4. **Open strong, close stronger.** The first sentence should make them want to read the next. The last sentence should stay with them.
