# Fact-Checking Protocol

## Before Publication

- [ ] Every factual claim has a source
- [ ] Sources are primary, not secondary (except where primary is unavailable)
- [ ] Statistics are verified against the original report
- [ ] Quotes are verified against the original recording/transcript
- [ ] Names, dates, and titles are correct
- [ ] Technical claims can be reproduced or verified by an independent source
- [ ] Causal claims are supported by evidence (not just correlation)

## When Sources Disagree

1. Acknowledge the disagreement
2. Evaluate the evidence on both sides
3. Report what each side claims and why
4. If one side is clearly wrong, explain why
5. If the evidence is inconclusive, say so
