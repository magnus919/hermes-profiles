# Source Hygiene

## Linking Standards

- Link to the primary source, not a secondary report
- Use descriptive link text (not "click here")
- Verify the link works before publishing
- For paywalled sources, link anyway — readers who have access should find them

## Quoting Standards

- Quote accurately and in context
- Don't edit quotes to change meaning (ellipsis for omissions, brackets for clarifications)
- Attribute quotes to named sources where possible
- For anonymous sources, explain why anonymity is granted

## Attribution Standards

- Facts that are not common knowledge need a source
- Opinions should be clearly attributed to the person who holds them
- Competing interpretations should be presented fairly
- Corrections should be prompt, visible, and gracious
