# Voice Reference

## Establishing Voice

- Know the audience before choosing the register
- Be consistent — voice shifts lose readers
- Write for the reader, not for yourself

## Common Registers

| Register | Characteristics | Best For |
|----------|----------------|----------|
| **Conversational** | First person, contractions, short sentences, personal anecdotes | Blog posts, opinion pieces, tutorials |
| **Analytical** | Third person, evidence-driven, measured claims, logical flow | Analysis pieces, case studies, technical reports |
| **Instructional** | Imperative mood, numbered steps, explicit outcomes | Documentation, guides, tutorials |
| **Narrative** | Scene-setting, character, tension, resolution | Story-driven pieces, case studies |

## Voice Consistency Checklist

- [ ] Opening register matches closing register
- [ ] No unexplained tone shifts between sections
- [ ] Vocabulary level is appropriate for the target audience
- [ ] Contractions are used consistently (either use them throughout or not at all)
- [ ] First/second/third person is consistent
