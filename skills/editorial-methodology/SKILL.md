# Editorial Methodology

Site-neutral editorial process for planning, drafting, and revising written content.

## The Editorial Lifecycle

```
PLAN → STRUCTURE → DRAFT → REVISE → VERIFY → PUBLISH
```

## Reference Files

| Reference | When to load |
|-----------|-------------|
| `references/voice-reference.md` | You need to establish or maintain a consistent voice for a piece |
| `references/article-structures.md` | You need to decide how to structure an article |
| `references/source-hygiene.md` | You need to link, quote, or cite external sources |
| `references/revision-protocol.md` | You need to revise a draft — structural edit, line edit, proofread |
| `references/fact-checking.md` | You need to verify factual claims before publication |
